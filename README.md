# Flask API Cache

Flask API Cache example with Docker

## Setup

* Create venv
```
py -m venv venv
```
* Activate venv
```
venv\Scripts\activate
```
* Install requirements.txt
```
pip install -r requirements.txt
```
## Docker

* Build
```
docker-compose -up --build
```
* Check Docker Container Status
```
docker ps
```
